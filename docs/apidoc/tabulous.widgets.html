class tabulous.widgets.filtering.FilterProxy(table: TableBase | None = None)[source]¶

Bases: object

A proxy of column-based DataFrame filtering.

This object makes filtering of table easier. For instance, to filter a table by a column, you can do: >>> table.filter[“a”] < 5 which is equivalent to >>> table.filter = lambda df: df[“a”] < 5
